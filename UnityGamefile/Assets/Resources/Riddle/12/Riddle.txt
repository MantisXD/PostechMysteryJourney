RiddleName "메인 수수께끼 #7. 성하경을 찾아라!"
Riddle "데이터 분석의 가장 기본은 데이터를 찾아내는 것이다. 그런데, 데이터는 저장 방법에 따라 찾아내기 어려울 수도 있고, 간단할 수도 있다. 포항공과대학교는 데이터를 객체화하여 저장하는 방법을 사용하는데, 메모리의 효율적 관리를 위하여 포인터를 배열에 저장하여 이를 새로 할당하는 방법을 사용한다. 그런데, 약간 문제가 생겼다. 
우리가 사용할 수 있는 클래스의 변수는 정해져 있는데, 그 변수들은 외부에서 참조할 수 없도록 한 것이다. 아래의 코드를 완성하여, 성하경의 현재 위치를 알아보자.
 
클래스 멤버 변수
String location
Bool in_out // 유저가 들어갔으면 true, 나왔으면 false
String name // 학생의 이름
Int student_number // 학생의 고유 번호

주의할 점
1) 학생의 고유 번호는 10년에 한 번 초기화가 되어 다른 학생 임에도 같은 고유 번호를 가지기도 한다.
2) 학생의 이름 또한 동명이인이 있을 수 있다. 하지만, 그 문제를 방지하기 위해 고유번호와 학생의 이름이 모두 같은 경우는 없었다고 한다.
3) 학생이 그 건물에 들어간 경우 in_out은 true가 출력되고, 나온 경우 out이 출력된다.
4) 자유롭게 함수를 추가할 수 있으나, cout을 사용할 경우 출력 결과를 확인 할 수 없으므로 추천하지 않는다.
5) Location의 종류는 location_all에 저장이 되어 있으니, 참고하도록 한다."
